### Variables
| **Variable**    | **Description**                            | **Type**   | **Default**   |
|-----------------|--------------------------------------------|------------|---------------|
| `environment`   | The environment of the resource group.      | `string`   | N/A           |
| `name`          | The name of the resource group.             | `string`   | N/A           |
| `location`      | The location of the resource group.         | `string`   | N/A           |
